Keyboard Summary ​

The Summary panel shows statistics about the current layout:

  • Keys by Size — Count of keys at each size (e.g., how many 1U, 1.25U, 2U keys). Toggle to By Size & Color to also group by key color — useful for planning keycap purchases.
  • Key Center Positions — Table of (X, Y) center coordinates for all keys, switchable between U and mm.
  • Keyboard Dimensions — Total width and height of the key cluster, in U or mm.
  • CSV Export — Download a CSV file with key center positions, dimensions, and rotation angles. Optionally include ghost and decal keys. Useful for external tooling or documentation.
